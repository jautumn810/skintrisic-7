# Personal Portfolio Website

This portfolio website project includes all required HTML, CSS, and JavaScript files for the assignment.

## Files Included

### HTML Files
- **resume.html** - Complete resume with all required elements
- **cover-letter.html** - Cover letter with required formatting
- **career-goals.html** - Career goals page with media and contact form

### CSS File
- **master.css** - External stylesheet with all required CSS selectors and properties

### Required Media Files (You need to add these)

You will need to add the following media files to complete the project:

1. **portfolio-image.jpg** (or .png, .jpeg, .gif)
   - Place in the same directory as the HTML files
   - Suggested: A professional photo or workspace image

2. **career-goals-audio.mp3** (or .wav)
   - Place in the same directory as the HTML files
   - Suggested: A brief audio recording about your career goals (30-60 seconds)

3. **career-video.mp4** (or .mov, .mpeg)
   - Place in the same directory as the HTML files
   - Suggested: A brief video introduction (1-2 minutes)

## How to Add Media Files

1. Obtain or create your image, audio, and video files
2. Ensure file names match exactly:
   - `portfolio-image.jpg` (or your preferred image format)
   - `career-goals-audio.mp3` (or .wav)
   - `career-video.mp4` (or .mov, .mpeg)
3. Place all files in the same directory as the HTML files
4. Test that all media files load correctly in your browser

## File Size Requirements

- Total submission: Maximum 200 MB
- Individual files: Maximum 16 MB each

## Testing

1. Open `resume.html` in a web browser
2. Test navigation between all three pages
3. Verify the contact form works
4. Check that all media files play correctly
5. Open browser console to verify JavaScript messages appear
